更新apt sudo apt-get update 安装wget sudo apt-get install wget zlib2 make 下载相应版本的gcc安装包（把下面的地址换成相应版本的gcc即可） wget https://ftp.gnu.org/gnu/gcc/gcc-12.2.0/gcc-12.2.0.tar.gz 解压安装包 tar -xzvf gcc-12.2.0.tar.gz 配置安装路径 sudo vim /etc/profile, 在最后添加 export PATH="/usr/local/gcc-12.2/bin:$PATH" 进入gcc源码目录 cd gcc-12.2.0/ 下载需要的配置 ./contrib/download_prerequisites 创建编译目录 cd .. mkdir temp_gcc9.2 && cd temp_gcc9.2 这里报错： configure: error: no acceptable C compiler found in $PATH 原因是没有安装gcc g++， 解决方法： apt-get install -y gcc g++ 再次执行后成功 编译安装 make && make install 链接 ln -s /usr/local/gcc-12.2/bin/gcc /usr/bin/gcc ln -s /usr/local/gcc-12.
